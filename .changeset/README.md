# Changesets

This folder contains the changesets used to manage versioning and publishing of the
`@novats/*` packages.

See https://github.com/changesets/changesets for documentation.

## Quick reference

- Add a changeset describing your change: `pnpm changeset`
- Bump versions based on accumulated changesets: `pnpm version-packages`
- Publish bumped packages to npm: `pnpm release`
